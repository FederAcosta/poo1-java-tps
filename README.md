# poo1-java-tps
Trabajos prácticos de Programación Orientada a Objetos en Java (POO1). Incluye ejercicios de clases, herencia y encapsulamiento.

## Proyecto: Sistema de Torneo de Fútbol

Simulación de un sistema de gestión de torneos con equipos, jugadores y partidos.

### Funcionalidades
- Creación de equipos y jugadores
- Registro de partidos
- Cálculo de estadísticas

### Tecnologías
- Java
- Programación Orientada a Objetos
